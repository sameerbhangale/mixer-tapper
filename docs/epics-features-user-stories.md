# Epics, Features, and User Stories

## Epic 1: Fast Beat Creation
### Features
- Tap-to-record beat pattern
- Metronome and count-in
- Quantization controls

### User Stories
- As a beat performer, I want to tap rhythm on the screen so I can create a loop in seconds.
- As a beginner, I want count-in before recording so I can start in time.
- As a user, I want quantization so slightly off taps are corrected.

## Epic 2: Live Sync with Singer
### Features
- Green Sync button
- Beat-cycle phase alignment
- Visual downbeat indicator

### User Stories
- As a performer, I want a single sync button so loop and singing align immediately.
- As a user, I want a visual downbeat pulse so I know when sync occurred.
- As a duo performer, I want repeatable sync behavior even after tempo changes.

## Epic 3: Mixer Output & Connectivity
### Features
- Bluetooth output selection
- Connection health monitor
- Reconnect flow

### User Stories
- As a performer, I want to choose my mixer Bluetooth device so output goes to speakers.
- As a user, I want reconnection prompts so a drop does not end the session.
- As a user, I want latency calibration so output stays musical.

## Epic 4: Session Control
### Features
- Layer mute/solo
- Volume mixing
- Save/load local sessions

### User Stories
- As a performer, I want to mute snare quickly so I can vary arrangement live.
- As a user, I want per-layer volume so I can balance with vocals.
- As a hobbyist, I want to save favorite grooves for future practice.

## Epic 5: Reliability & Accessibility
### Features
- Offline-first operation
- Large controls / high contrast mode
- Crash recovery

### User Stories
- As a musician, I want app to work without internet during gigs.
- As a user in low light, I want high contrast controls for easy use.
- As a user, I want session auto-recovery after interruption.
